# raw2pts
Raw file 2 point cloud file


## Compile

``` shell
$ gcc raw2pts.c -o raw2pts
```

## Run

``` shell
$ raw2pts demo.raw demo.txt
```

* demo.raw: <http://gnat.qiniudn.com/raw2pts/demo.raw>
* demo.txt: <http://gnat.qiniudn.com/raw2pts/demo.txt>
